# SlimeGameWeb3

This project is a blockchain-based mini-game on the Polygon network.

## Environment Setup

Navigate to the Slime directory.

In the terminal, enter:

yarn install



or

npm install

vbnet


To run the project on localhost:3000, use:

npm start

ruby
